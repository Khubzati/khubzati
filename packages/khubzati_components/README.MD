# Khubzati Components

This package contains shared components and utilities used across the Khubzati project.

## Getting Started

### Prerequisites

- [Flutter](https://flutter.dev/docs/get-started/install) installed on your machine.

### Setup

1. **Navigate to the package directory**:

   ```bash
   cd packages/khubzati_components
   ```

2. **Install dependencies**:

   ```bash
   flutter pub get
   ```

## Development

### Common Commands

- **Analyze Code**:

  ```bash
  flutter analyze
  ```

- **Clean Project**:

  ```bash
  flutter clean
  ```

- **Generate Code**:

  Use build_runner to generate code for the project.

  ```bash
  flutter pub run build_runner build --delete-conflicting-outputs
  ```
